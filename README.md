# Privacy Auditor Web

Privacy Auditor Web is a simple browser privacy audit tool made with HTML, CSS, and JavaScript.

It shows what kind of information a normal website can read from a user's browser, like device information, browser signals, and a basic tracking risk score.

## What it does

- Shows basic device information
- Displays browser signals
- Gives a simple tracking risk score
- Explains how browser fingerprinting can work without cookies

## Why I made it

I built this project to learn more about browser privacy and how websites can collect small details from a device. It is not meant to attack anything. It is only for learning and awareness.

## Built with

- HTML
- CSS
- JavaScript
