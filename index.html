Browser Privacity Audit

I made this project to see much information a normal website can read from a browser. It does not attack anything or break into anything. It only collects basic data that browsers usually expose to websites.

This is for learning and awareness. The point is to understand browser fingerprinting and how tracking can work even without cookies.

How to Project Yourself

  • Use a privacy-focused browser: Firefox with strick tracking protection or Brave can reduce a lot of tracking.
    • Install uBlock Origin: This blocks many ads, trackers, and suspicius scripts before they load.
  • Do not install random extensions: Extensions can make your browser more unique and sometimes collect data too.
  • Use a trusted VPN when needed: A VPN can hide your IP from website, but it does not make you fully anonymus.
  • Keep your browser updated: Old browsers can have privacy problems and security bugs.

Disadvantages and Possible Attacks

  • Fingerprinting: Website can combine your screen size, timezone, plataform, canvas rendering, and other details to recognize your browser again.
  • IP tracking: Your public IP can reveal your internet provider and approximate location.
    • Extension tracking: Some browser extensions can collect browsing behavior if they are not trustworthy.
  • Canvas fingerprinting: Small diferences in graphics rendering can create a unique browser signature.
